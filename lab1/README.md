# Лабораторная работа: Алгоритмы цифровой обработки сигналов

### **Цель работы**
Для заданных дискретных сигналов **x(t)** и **y(t)** (или специально подобранных музыкальных файлов) выполнить следующие задания в соответствии с выданным вариантом.

---

## **Задания**

### **1. Операция свёртки**
Реализовать операцию свёртки двух сигналов.

### **2. Операция корреляции**
Реализовать операцию корреляции двух сигналов.

### **3. Преобразование Фурье**
Для сигналов **x(t)**, **y(t)** и результата свёртки выполнить:
*   Прямое преобразование Фурье.
*   Обратное преобразование Фурье.

### **4. Построение графиков**
Построить графики:
*   Амплитудного спектра сигналов.
*   Фазового спектра сигналов.
*   Сигналов во временной области.

### **5. Экспериментальная проверка**
Проверить корректность схем вычисления свёртки и корреляции (согласно рисункам 15 и 16 методического пособия).

### **6. Анализ результатов**
Сравнить полученные результаты при разных параметрах и оценить эффективность использованных алгоритмов.
